HTML/CSS Reference Sheet

HTML

Basic HTML tags
tag name description
<html> html Basic opening HTML tag used to open a HTML file
<head> head area for links to the css page and the title
<body> body area for the main content of the webpage development
<title> title title of website, will display in the tab bar
<Nav> nav Nav bar for basic navigation around page
<footer> footer footer tag to allow for bottom of page. contact info usualy goes here
<link> Link connects html to css pages. use href and rel to connect
<a> a used for links. The text of the link is the clickable part usually outlined in blue.
<img> image Image tag that will be used for images. src will be for link to the image in your files.
<ul> unordered list list tag that has no specific order
<ol> ordered list list that has a specific order
<li> list item specific item on a list
<div> division generic term used for seperating areas of code. Uses an identifier to seperate it from others.
<section> section Term used to seperate different areas of html code
Common Types of Identifiers
tag name description
id identification identifier used to identify seperate sections of a div tag. Inside of basic html tag. Uses ".id" form when used in css. (For example: id="css", then .css)
class class tag identifier used to identify seperate classes inside of a div tag. Inside of a basic htm tag. Uses "#class" form when used in css. (For example: class=taco, then #taco)

All About The Tables
tag name description
<table> table opening tag for the beginning of any table
<thead> table head Like the head tag in the html tag. This is where you put the information for the name of the columns that are seen above any table
<tbody> table body the main body of the table. Where you put the rows and the table data
<caption> caption this is for the caption of the tablke or where you describe the table for instance.
<tr> table row this is used to set up the row in each table
<th> table row head This is used to put the name that the rest of the row describes. Also row name
<td> table data data for the table is placed here.
